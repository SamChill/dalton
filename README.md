## Installation
Compile and run like so:

    sudo apt install build-essential cmake xorg-dev libglu1-mesa-dev libboost-filesystem-dev
    git clone https://github.com/SamChill/dalton.git
    cd dalton
    make -j 4
    ./build/dalton
